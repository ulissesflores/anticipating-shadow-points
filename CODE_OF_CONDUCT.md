# Code of Conduct

This project adopts the **[Contributor Covenant version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct.

The canonical text is hosted at the URL above and is incorporated here by reference. Read it in full before participating.

## Short version

We pledge to make participation in this project a positive experience for everyone, regardless of background, identity, or experience level. We commit to fostering an open, welcoming, and respectful community.

## Reporting

Concerns about behavior can be reported confidentially:

- **Email**: c.ulisses@gmail.com (subject: `[asp-conduct]`)
- **GitHub**: contact [@ulissesflores](https://github.com/ulissesflores) directly via the contact form at [ulissesflores.com](https://ulissesflores.com)

All reports are reviewed promptly and the privacy of the reporter is respected.

## Enforcement

Maintainers follow the **Community Impact Guidelines** from Contributor Covenant 2.1 when responding to behavior that does not align with this code:

1. **Correction** — private clarification of the issue.
2. **Warning** — formal warning; further engagement may be restricted.
3. **Temporary ban** — temporary restriction from project spaces.
4. **Permanent ban** — permanent restriction from project spaces.

The exact criteria for each level are documented in the Contributor Covenant 2.1 text linked above.
