# Translations

ASP documentation is available in five languages. The English version is the canonical source; translations are kept structurally parallel but may lag minor updates by a few hours/days.

| Language | Author / Reviewer | Status |
|---|---|---|
| 🇬🇧 [English](../) | Native | Canonical source |
| 🇪🇸 [Español](README.es) | AI-assisted | Native review welcomed via PR |
| 🇧🇷 [Português](README.pt) | Native (Brazilian) | Author-reviewed |
| 🇮🇹 [Italiano](README.it) | AI-assisted | Native review welcomed via PR |
| 🇮🇱 [עברית](README.he) | AI-assisted (RTL marked) | Native review welcomed via PR |

> [!NOTE]
> If you're a native speaker of ES, IT, or HE and notice translation issues, please open a PR. We prefer accurate translations over silent corrections — your authorship attribution will be added.
